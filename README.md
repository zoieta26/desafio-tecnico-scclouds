# desafio-tecnico-scclouds
Resolução dos desafios técnicos de lógica (Fibonacci e Números Primos) em Python para a vaga de estágio.
Este repositório contém a resolução dos desafios de lógica para o processo seletivo da **SC CLOUDS**

### Tecnologias Utilizadas
* **Linguagem:** Python 3.14
* **Versionamento:** Git/GitHub

### Estrutura do Projeto
* `fibonacci.py`: Funções para o cálculo da sequência de Fibonacci (Linear e Recursiva).
* `primos.py`: Funções para a identificação de números primos (Linear e Recursiva).
